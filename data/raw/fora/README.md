# FORA Corpus

The FORA transcript files are not included in the public release of this repository while redistribution approval is pending.

The source corpus is maintained upstream at:

https://github.com/schropes/fora-corpus

Please request or obtain FORA data through the upstream corpus repository and its access process. This project releases FORA-derived annotation labels separately under `data/release/fora/`, without bundling the full FORA conversation transcripts.
